# Visual French-English Dictionary — Assets

## images/
Place word illustration PNGs here.
Naming convention: `{english_word}.png`  
Example: `dog.png`, `apple.png`, `table.png`

Recommended size: 512×512 px, transparent background.

## audio/
Place pronunciation MP3 files here.
Naming convention: `{english_word}.mp3`
Example: `dog.mp3`, `apple.mp3`, `table.mp3`

You can generate these with text-to-speech tools such as:
- Google Cloud Text-to-Speech (French voice: fr-FR-Standard-A)
- Amazon Polly (Mathieu / Céline)
- ElevenLabs

## data/
Reserved for future JSON export/import of the word database.
